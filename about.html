This is a prototype site for a GitHub & JavaScript-powered incubator. To add a project to labs, create a repository for it under the OKFN account on GitHub. Then simply upload a .labs_project metadata file to its root directory. 

labs_project({
    "image" : "http://link_to_image.gif",
    "category" : "Category Of Project",
    "name" : "My Project Name",
    "status" : "Active (or not)",
    "strapline" : "A few words to explain it",
    "description" : "A lengthy description in paragraphs",
    "link" : "http://link.to.homepage",
    "imagename" : "not_used"
});
